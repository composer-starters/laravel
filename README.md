## About the Starter Kit

This is [NiftyCo's](https://aniftyco.com) opinionated Laravel starter kit. It is the defacto starting point for any
NiftyCo project built on Laravel, pre-configured with the tooling, conventions, and defaults we reach for on every
project.

What's included out of the box:

- [Laravel 13](https://laravel.com) on PHP 8.5.
- [Pest 4](https://pestphp.com) for testing, with the Laravel plugin pre-wired.
- [Pint](https://laravel.com/docs/pint) for PHP code style.
- [Vite 8](https://vitejs.dev) and [Tailwind CSS 4](https://tailwindcss.com) for the frontend, with TypeScript ready to
  go.
- Node 24 and npm 11.3 pinned via `engines`.

## Getting Started

Scaffold a new project with the [Laravel installer](https://laravel.com/docs/installation):

```shell
laravel new my-app --using starter/laravel
```

Replace `my-app` with whatever you want to name your project.
